## Technical Constraints

- **Single file**: Everything in one `index.html` (HTML + CSS + JavaScript)
- **No external dependencies** except CDN-hosted libraries (Three.js, etc.)
- **Mobile-first**: Touch controls, viewport meta tags, responsive design
- **Performant**: Should run smoothly on a phone
